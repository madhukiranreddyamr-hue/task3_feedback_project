# Feedback System (Task3)

## Description

This project is developed using Python and Flask.
It allows users to submit feedback which is stored in a MySQL database.

## Features

* User Feedback Form
* Store feedback in MySQL database
* Success page after submission
* Frontend using HTML, CSS, JavaScript

## Technologies Used

* Python
* Flask
* MySQL

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Configure MySQL database in db_config.py

3. Run the application:
   python app.py
